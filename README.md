# 🪨📄✂️ Rock Paper Scissors Game

A fun **Rock Paper Scissors mini project** built using **HTML, CSS, and JavaScript**, focused on strengthening **logic building, scenario-based thinking, DOM manipulation, score tracking, and interactive UI skills**.

This project was built after completing a Tic Tac Toe game as the next step in improving JavaScript fundamentals through mini projects.

---

## ✨ Features

* 🪨 Stone, 📄 Paper, ✂️ Scissors choices
* 🤖 Random computer move generation
* 🧠 Win / Lose / Draw game logic
* 📊 Dynamic score tracking
* 🔄 Reset game functionality
* 🎨 Interactive UI with hover effects
* 📱 Beginner-friendly responsive layout
* ⚡ Instant result updates

---
